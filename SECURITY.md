# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the BGPT MCP API, please report it responsibly:

**Email:** [contact@bgpt.pro](mailto:contact@bgpt.pro)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and work to resolve the issue promptly.

## Scope

This policy covers:
- The BGPT MCP API endpoint (`https://bgpt.pro/mcp/sse`)
- This repository's documentation and example code

## Disclosure

We follow responsible disclosure. Please do not publicly disclose vulnerabilities until we've had a chance to address them.

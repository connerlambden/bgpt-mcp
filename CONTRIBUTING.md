# Contributing to BGPT MCP API

Thanks for your interest in contributing! This repo contains documentation, examples, and configuration for the BGPT MCP API. The server itself is hosted at [bgpt.pro/mcp](https://bgpt.pro/mcp/).

## How to Contribute

### Report a Bug

If you encounter an issue with the BGPT MCP API:

1. Check [existing issues](../../issues) to avoid duplicates
2. Open a [new issue](../../issues/new?template=bug_report.md) with:
   - What you expected to happen
   - What actually happened
   - Your MCP client (Claude Desktop, Cursor, etc.)
   - Steps to reproduce

### Request a Feature

Have an idea for improving BGPT? Open a [feature request](../../issues/new?template=feature_request.md).

### Improve Documentation

Found a typo, unclear instruction, or missing example? PRs are welcome:

1. Fork the repo
2. Create a branch (`git checkout -b fix/typo-in-readme`)
3. Make your changes
4. Submit a pull request

### Add Examples

We welcome example scripts showing how to use BGPT with different tools and languages. Add them to the `examples/` directory.

## Code of Conduct

Be respectful, constructive, and collaborative. We're building tools for science.
